# Sentinel Ledger

Monorepo simple:

- `backend/`: API Express + Socket.IO
- `frontend/`: Next.js (pages router) + Tailwind
- `supabase/`: `schema.sql`

## Local

Backend:

1. `cd backend`
2. Copia `.env.example` a `.env` y rellena variables
3. `npm install`
4. `npm run dev`

Frontend:

1. `cd frontend`
2. `npm install`
3. `npm run dev`

## Supabase

Ejecuta `supabase/schema.sql` en tu proyecto Supabase (SQL editor).
